# Documentacion de IA en TaskFlow

Esta carpeta recoge el trabajo de la segunda parte del proyecto: como he usado herramientas de IA para analizar, refactorizar, documentar y mejorar TaskFlow.

## Archivos

- `comparativa-asistentes.md`: comparacion entre ChatGPT, Claude, Cursor y GitHub Copilot.
- `prompts.md`: prompts usados y versiones mejoradas.
- `flujo-cursor.md`: notas sobre el flujo de trabajo con Cursor.
- `mcp-refactor.md`: refactorizacion, JSDoc y uso de MCP.
- `experimentos.md`: pruebas de prompting, busqueda, edicion y resultados.
- `reflexion-final.md`: beneficios, riesgos y conclusion personal.

## Objetivo

El objetivo no es solo decir que se ha usado IA, sino dejar evidencia de que se ha integrado en el flujo de desarrollo: pedir explicaciones, comparar respuestas, detectar errores, generar funciones, documentar decisiones y mejorar el codigo real del proyecto.
